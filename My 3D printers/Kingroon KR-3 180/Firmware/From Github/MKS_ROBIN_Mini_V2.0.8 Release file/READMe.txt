1、According to the content that needs to be updated, copy the following file to the SD card:

 (1)、update the config file：robin_mini_config.txt
 (2)、update images：         mks_pic
 (3)、update font：           mks_font
 (4)、update TFT firmware：   Robin_mini.bin
 (5)、update Wifi firmware：  MksWifi.bin

2、When using the SD card to update the configuration file, you need to change the configuration file name to: robin_config.txt

3、When configuring the red version, you need to change the background color of the configuration file to 0x494949

4、Configure the language of  Win8 version need to set : cfg_multiple_language:0

5、The firmware of Robin_Mini_V2.0.7 can only be used in Robin_Mini_V2.0 hardware.

6、V2.0.8 Modifications:
 (1)、Fix the bug that read Large capacity sd card data error sometimes.
